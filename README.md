# ⚖️ LexisVault

🚧 Projeto em desenvolvimento

Sistema jurídico para gerenciamento de processos, usuários e autenticação segura.

## 🧠 Funcionalidades atuais
- Autenticação com JWT
- Login de usuários
- Estrutura backend segura

## 🛠️ Arquitetura
- Backend: Java + Spring Boot
- Frontend: React + TypeScript
- Banco: PostgreSQL

## 🔄 Em desenvolvimento
- Cadastro de processos
- Dashboard
- Integração completa frontend/backend

## ▶️ Como rodar

### Backend
./mvnw spring-boot:run

### Frontend
npm install
npm run dev
